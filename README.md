# 事件配点ゲーム

## セットアップ
1. このフォルダを用意
2. ブラウザで `index.html` を開く（ローカルでOK／サーバ不要）
3. 任意で効果音ファイルを同ディレクトリに配置  
   - `se_correct.mp3`, `se_wrong.mp3`, `se_start.mp3`

## 操作・流れ
- 難易度（イージー / ノーマル / ハード）を選択して開始
- 最初のトークンが事件係前に表示されてから **3,2,1 カウントダウン** 後に動き出す  
  2体目以降はカウントなし
- 矢印のある交差点をクリック／タップで進行方向を時計回りに切替
- トークンは **交差点 → ライン上 → 次の交差点** と移動
- 番号に対応する部屋（1〜8）へ正しく誘導するとスコア加算
- 誤配・滞留でライフ減少。ライフ0または時間切れで終了

## レイアウト
- 右側サイドバーにタイトル／スコア／時間／ライフを縦配置
- メインエリアはキャンバスが全画面表示（PC/スマホ対応）
- 縦向きスマホでは向き警告を表示

## 難易度と終盤加速
- イージー: 最大トークン2、スポーン 9→8→7秒、終盤加速は緩やか
- ノーマル: 最大トークン3、スポーン 8→7→6秒、終盤で移動が速くなる
- ハード: 最大トークン3、スポーン 7→6→5秒、終盤でさらに速くなる
- 終盤に向かってライン移動時間が短縮（イージーは控えめ、ノーマル/ハードは強め）

## 仕様メモ
- 部屋は8室（特別室なし）
- トークン重なり防止 & 矢印が隠れないよう描画順を調整
- 効果音は `showFeedback`（正配/誤配/滞留）とカウントダウン終了で再生

## トラブルシュート
- 動かない: `index.html` を開き、難易度を選択して開始
- 表示崩れ: ブラウザのズームを100%にするか、リサイズして再読み込み
- 音が出ない: mp3ファイル設置とブラウザの音量設定を確認
