# 事件記録ルーティングゲーム

## セットアップ
1. リポジトリ（またはこのフォルダ）を用意
2. ブラウザで `index.html` を開く  
   - ローカルファイルとして開くだけで実行できます（サーバ不要）

## 操作方法
- ページ表示後、難易度（イージー / ノーマル / ハード）を選択して開始
- 矢印がある交差点をクリック／タップすると進行方向が時計回りに切り替わります
- トークンは **交差点 → ライン上 → 次の交差点** の順にゆっくり移動します
- 事件係の前に次に来るトークンが表示されます（番号＝配送先の部屋）
- 右側と下側にある 1〜8 の部屋へ正しく誘導するとスコアが加算されます
- 誤配・滞留でライフが減り、ライフが 0 か時間切れで終了

## 画面構成
- ヘッダー: スコア / 時間 / ライフ
- キャンバス: グリッド、矢印、トークン、部屋、事件係
- リザルト: スコアと統計表示、リスタートボタン

## 主な仕様
- 部屋は 8 部屋（特別室なし）
- トークン生成は難易度ごとに間隔と上限を調整
- ライン移動は約 1.2 秒／マスでのんびり進行
- トークン重なりを防止し、矢印が隠れないよう再描画順を調整
- PC／スマホ両対応、縦向き時は警告を表示

## トラブルシュート
- 何も動かない: ブラウザで `index.html` を開き、難易度を選択して開始してください
- 表示が崩れる: ブラウザのズームを 100% にするか、画面をリサイズして再読み込み
- 反応しない: タップ／クリックがキャンバス上であることを確認してください
